Данный скрипт работает только при установленных модулях HPEOACmdlets. Если их нет можно прогнать скрипт установки данных модулей - Install-PackageProvider Nuget.ps1
Имеется файл Blades.xlsx, в каждом листе которого в табличном виде отображается инфа лезвий имеющихся в данном ЦОД-е блейд-шасси.
Данный скрипт по входящему параметру (названию файла - Blades.xlsx) ищет в нём IP адреса OA и пытается подрубиться по данному IP к блейду. Проивзодит выгрузку основных характеристик имеющихся лезвий - модель лезвия, серийник, проц (количество и модель), оперативка (общий объём), версия iLo. Затем все выгруженные данные вносятся в файл.
